# Todo Client (React + Vite)

Este es el frontend para la Todo API, construido con React, Vite y Tailwind CSS.

## Instalación

1.  Asegúrate de estar en la carpeta `todo_client`:
    ```bash
    cd todo_client
    ```

2.  Instala las dependencias:
    ```bash
    npm install
    ```

## Ejecución

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`.

## Estructura

*   `src/components`: Componentes reutilizables (AddTodo, TodoItem).
*   `src/services`: Lógica de conexión con la API.
*   `src/lib`: Utilidades (Tailwind merge).
*   `src/App.jsx`: Componente principal.
